
Bowmaster Clone (Mobile Optimized)

Files required:
- index.html
- archer.png -> Archer character
- target.png -> Moving target
- audio1.mp3 -> Background music
- audio2.mp3 -> Hit sound

Instructions:
1. Open index.html in browser.
2. Drag/touch to aim bow up/down.
3. Release/click to shoot arrow.
4. Hit target to increase score/level.
5. Each hit increases target speed.
6. Works on mobile and desktop.
